Oracle Integration Cloud Specialist
Our client, specialised in language education, is looking for a Junior Oracle Cloud Integration Specialist.
As a Junior Oracle Integration Cloud Specialist you will be 
Responsible for developing and maintaining Oracle Integration Cloud (OIC) integrations, Application Integration, Visual Builder, and Insight Analytics. Focus will be on application integration between Oracle ERP Cloud and other 3rd party systems, automate manual processes and error handling for the current existing ones.
 The ideal candidate should have:
• Experience with Oracle Integration Cloud, or willingness to work in it
• SQL knowledge
• Data modeling experience
• Data migration experience would be nice to have
