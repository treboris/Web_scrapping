Rendszerszervező
Partnerünk telematika területén a leginnovatívabb magyar vállalat. Számtalan izgalmas hazai és nemzetközi projekten dolgoznak, amelyekhez most új munkatársat keresnek.
Feladatok:
Szakértői erőforrásbecslés és a feladat leszállítható egységekre bontása
Átfogó tervdokumentációk készítése
Követelmény specifikáció (PRD) készítése, valamint funkcionális (FSD) specifikáció készítése a vezető fejlesztőkkel közösen
Fejlesztés koordinálása - scrum masterrel és fejlesztői csapattal való együttműködés
Release folyamat koordinálása szoros együttműködésben az üzemeltetőkkel és fejlesztőkkel
Rendszerarchitektúra tervezésének támogatása
Eredmény dokumentációk készítése
Éles rendszerek folyamatos üzleti- és technikai támogatása
Elvárások:
Felsőfokú végzettség
Legalább 5 éves IT szervezői tapasztalat
UML vagy BPMN - folyamat modellezésben és rendszertervek készítésében szerzett több éves tapasztalat
Integrált, nagy rendelkezésre állású, nagy terhelésű rendszerek tervezésében szerzett tapasztalat
API teszteszközök ismerete (pl: SoapUI; Postman)
Ticketing rendszer ismerete (pl. Jira)
Verziókezelők ismerete (pl. SVN, Git)
SQL nyelv magas szintű ismerete, valamelyik SQL fejlesztőeszköz készség szintű ismerete (pl. SQL Developer, Toad)
Adatbázis, interfész és UI felület tervezésében tapasztalat
Kiváló problémafelismerő és megoldó készség
Eredményorientált üzleti gondolkodás
Rendszerszintű gondolkodásmód
Önálló és csapatban való munkavégzés
Ajánlatunk:
Változatos hazai és nemzetközi projektek a legmodernebb technológiákra alapozva (Enterprise környezet)
Üzletileg kritikus, magas rendelkezésre állású IT rendszerek fejlesztése (és üzemeltetése)
Komplex, felelősségteljes munka a rendszerek fejlesztésében
Szakmai képzéseken, konferenciákon való részvételi lehetőség
Stabil vállalat, fiatalos munkahely
Dinamikus, szakmailag tapasztalt, széles látókörű kollégák, csapat, vezetőség
Jól megközelíthető, felszerelt modern iroda
Versenyképes bérezés
Előnyök:
Felsőfokú végzettség informatikai vagy gazdaságinformatikai területen
Agilis módszerek és folyamatok aktív használata
Tervezési tapasztalat kapcsolódva az alábbi rétegekhez: 
Oracle vagy más RDBMS adatbázis 
Java EE backend 
XML/JSON alapú kommunikációban szerzett tapasztalat 
Angular frontend 
Alábbi eszközök ismerete: 
EA 
XMLSpy vagy hasonló eszköz 
Confluence 
IDE eszközök (Pl.: Eclipse, IntelliJ) 
Egyéb tapasztalatok: 
rendszer architektúrák ismerete (monolit, SOA, microservice) 
NoSQL adatbázisok ismerete (pl. MongoDB, etcd) 
Automata tesztelési eszközök ismerete (pl. JMeter) 
Logok elemzésében szerzett tapasztalat Linux környezetben (pl. less, tail, grep…) 
CI toolok és folyamat tapasztalat (pl. Jenkins, GO) 
Virtualizációval, konténerizációval kapcsolatos tapasztalatok 
Kód olvasás valamilyen objektum orientált, vagy script nyelven
