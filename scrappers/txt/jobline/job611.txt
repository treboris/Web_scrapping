Senior Cloud/Big Data Engineer
Your new company
E-commerce start-up on a quest to develop a market leading e-commerce platform from ground zero.
Your new role
o Setup the Azure Data Platform for the Enterprise
o Setup the Middleware domain for the Enterprise (initially Istio Service Mesh / Azure Service Bus and Apache Kafka / Azure Event Hub)
o Develop Data Integration / Transmission / Replication Scenarios
o Develop Data Capture / Processing / Transformation / Import Scenarios
o Develop Service Provisinoing / Invocation / Orchestration Scenarios
o Develop Event Publication / Subscription / Processing Scenarios
o Support the Capability Maturity Model development and roadmap execution of the Middleware Domain
o Support the Capability Maturity Model development and roadmap execution of the Data Domain
What you'll need to succeed
o Experience with Data Modeling, Master Data Management, Data Lifecycle, Data ETL, Data Governance, Data Protection areas
o Experience with Data Persistance, Staging Technologies, DWH, Data Mart, Data Lake
o Experience with Big Data Technologies (Splunk, Hadoop, MongoDB, Kafka)
o Experience with Azure Data Platform (Ingest Pipelines, Azure DataLake Gen2, Azure SQL)
o Experience with Middleware, Integration, Complex Event Processing Technologies is a plus
o Experience with Mulesoft, Azure EventHub, Azure ServiceBus is a plus
o Optionally have Architecture Experience in Event Driven Architectures, Service Oriented / Microservices Architectures, Integration Patterns, UML 2.0, B2B Integration
What you'll get in return
o Competitive salary
o Annual bonus
o Flexible home office policy
