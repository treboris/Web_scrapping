Information about the position Job description, responsibilities and dutiesThe IT department of e-commerce web service based in Slovakia is looking for a Senior PHP developer. 
Our company supplies furniture to Slovakia, Czech Republic, Hungary, Romania and Croatia through such websites as novynabytok.sk, zondo.hu and several more. The office, based in Bratislava, has a relaxed professional environment with English, Slovak and Russian speakers.
The job entails developing the customer facing website, back office staff portal and maintaining a bunch of internal systems, made in PHP, MySQL, PostgreSQL, CouchDB, JavaScript (Mainly React), React Native and a few more minorly used technologies. 
Candidates should be decisive and able to work independently within a team. They should also have experience of building web portals including maintaining databases, organizing and deploying code and implementing new features and technologies.
Requirements:
- 3+ years working experience with PHP(7) and SQL.
- Document-based databases (We use CouchDB).
- HTML/CSS/JS (HTML5, CSS3, Vanilla).
- Experience with version control (Git).
- No fear to get your hands dirty with legacy spaghetti (we still have quite a lot of it)
- Ability to communicate in English, the interview will be in English.
Experience in these fields would be a great plus:
- Linux server administration.
- ES6, React
- CI/CD
- DDD
- Unit testing (PHPUnit, Jest, Enzyme)
- Object Oriented design patterns.
Nice-to-have skills:
- Module bundling (Webpack).
- Task systems (e.g. Jira, we use Youtrack).
- Templating modules for PHP (Mustache).
- Message Brokers (we use RabbitMQ)
- Mobile development with React Native.
