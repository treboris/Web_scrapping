ElvárásokAkit keresünk: ·         Tud felelősséget vállalni, élvezi az összetett környezet nyújtotta izgalmakat ·         Szeretne felhasználó-támogatással foglalkozni ·         Középfokú angol nyelvtudással rendelkezik FeladatokGyakornokként az alábbi feladatokban vehetsz részt: ·         Céges infrastruktúra üzemeltetése, karbantartása ·         Felhasználók támogatása ·         Gyakorlati tapasztalat szerzése az alábbi területeken:          Windows Server 2008 R2, 2012, 2016, 2019, Ubuntu 12.04, 16.04, 18.04 és 20.04 LTS Server, VMware ESXi, Photon OS, Centos 5, Debian 7, 8          Microsoft Exchange Server 2007, 2013          SharePoint, Apache, Tomcat, Nginx, MySQL, MSSQL, PostgreSQL, Oracle, Zabbix, TFS, SVN, Git, Jira, Windows PowerShell, shell scipt-ek; hálózati ismeretek, VPN; ElőnyökEz a program egyben szakmai gyakorlatod, TDK dolgozatod, vagy diplomamunkád témája is lehet. Lehetőséged nyílik az rendszerüzemeltetői folyamatokat közvetlen közelről megismerni. Beilleszkedésedről, valamint a szakmai segítségről és fejlődésedről rendszergazdáink gondoskodnak. Olyan embert keresünk, aki tud felelősséget vállalni, élvezi az összetett környezet nyújtotta izgalmakat, valamint motiválja, hogy önálló, kreatív gondolataival és ötleteivel hozzájárulhat egy szervezet üzleti sikereihez. A programban való részvétel nem jár bérezéssel, de a program végeztével legjobban teljesítő gyakornokainknak állást kínálunk. Célunk, hogy a gyakornoki programmal egy hosszú távú együttműködés kezdődjön el közöttünk. Ha kezdetben egyetem mellett szeretnél dolgozni, részmunkaidőben és teljes munkaidőben is van erre lehetőséged!
