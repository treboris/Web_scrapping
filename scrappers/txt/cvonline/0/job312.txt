Senior Azure Cloud Architect 
Country: Hungary (+36) • Budapest
Description:
We are looking for a Senior Azure Cloude Architect for a new venture of one of the region's most established multinational company.
Duties:
• Lead the foundation of the Cloud Architecture & Governance for the SME Corporation
• Support the Azure DataPlatform Architecture design
• Support the Azure DevOps Architecture design
• Support the Azure RunTime Infrastructure design for the core business enabler Marketplace Platform in Azure Kubernetes Services and other native Azure Cloud Services
• Support the Azure EventHub/DataServices Integration Architecture design
• Design and Prepare the Policy Environment within Azure Cloud
• Support the Azure RunTime Infrastructure design for the core business enabler Marketplace Platform in Azure Kubernetes Services and other native Azure Cloud Services
• Support integration initiatives and related Solution Architecture in/outbound from Azure Cloud
• Support SaaS ICT Architecture and Implementation in Azure
• Design and Prepare the Policy Environment within Azure Cloud
• Support integration initiatives and related Solution Architecture in/outbound from Azure Cloud
• Work together with the Chief Security Architect / Virtual CISO on Cloud Security Topics
• Support implementation of mitigations per the Business Continuity Plan
• Support the preparation of an IT Policy for the Enterprise
Requirements and skills:
• Experience in setting up and operating an Azure Cloud environment for a SME
• Expert within Azure IaaS, SaaS Cloud Services and Architecture
• Experience with Azure Networking services, Azure Security and Identity services, Azure Integration services, Azure SQL Database, SQL Server, SQL Server IaaS, firewalls, Web App proxies, Bash, BGP, Chef, Puppet, PowerShell scripting, Azure Monitor and Application Insights
• Experience with configuring, deploying and operating public cloud services (e.g. Azure, AWS, GCP)
• Expertise in infrastructure as code (e.g., cloud formation, ARM, Terraform, Ansible, Chef, Puppet)
• Basic familiarity with network features • e.g. cloud network topology, BGP, routing, web (REST) APIs
• Experience with configuring, deploying and operating public cloud services (e.g. Azure, AWS, GCP)
• Awareness in open source tools & scripting language (e.g. Python, powershell, Shell)
• Deep understanding of software development lifecycles and cloud economics, incl. knowledge of consumption-driven TCO
• Expertise in infrastructure as code (e.g., cloud formation, ARM, Terraform, Ansible, Chef, Puppet)
• High availability engineering experience (region, availability zone, data replication clustering)
• Good knowledge of security implications of public & private cloud infra design
• Experience with Cloud Automation and Infrastructure as Code
• Microsoft Certified Azure Architect is a plus
Command of languages:
Good command of English
Client company profile:
Our client company is a new venture for one of the region's most established multinational company. Strong backing and commitment from the Shareholders
What they offer:
• Competitive Compensation Package and Incentive System (13th and 14th Month salary bonus + Annual financial bonus based on company goals)
• 3 days home office/week, flexible working hours
• Less Regulation, Fast Moving Agile Organization
• State of the Art Technologies and Capabilities relying on in-house developed Products and 3rd Party Cloud SaaS / PaaS
• Lots of room for career growth
• Efficient Business Operations, lean Processes and low level of Bureaucracy
• Regional ambition, potential international expansion ahead
