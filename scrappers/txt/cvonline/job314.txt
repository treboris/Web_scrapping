ÁLLÁSHIRDETÉS FELADÁS
Senior Big Data Engineer / DWH / Middleware SME 
Country: Hungary (+36) • Budapest
Description:
We are looking for a Senior Big Data Engineer for a new venture of one of the region's most established multinational company.
Duties:
• Setup the Azure Data Platform for the Enterprise
• Setup the Middleware domain for the Enterprise (initially Istio Service Mesh / Azure Service Bus and Apache Kafka / Azure Event Hub)
• Develop Data Integration / Transmission / Replication Scenarios
• Develop Data Capture / Processing / Transformation / Import Scenarios
• Develop Service Provisinoing / Invocation / Orchestration Scenarios
• Develop Event Publication / Subscription / Processing Scenarios
• Support the Capability Maturity Model development and roadmap execution of the Middleware Domain
• Support the Capability Maturity Model development and roadmap execution of the Data Domain
Requirements and skills:
• Experience with Data Modeling, Master Data Management, Data Lifecycle, Data ETL, Data Governance, Data Protection areas
• Experience with Data Persistance, Staging Technologies, DWH, Data Mart, Data Lake
• Experience with Big Data Technologies (Splunk, Hadoop, MongoDB, Kafka)
• Experience with Azure Data Platform (Ingest Pipelines, Azure DataLake Gen2, Azure SQL)
• Experience with Middleware, Integration, Complex Event Processing Technologies is a plus
• Experience with Mulesoft, Azure EventHub, Azure ServiceBus is a plus
• Optionally have Architecture Experience in Event Driven Architectures, Service Oriented / Microservices Architectures, Integration Patterns, UML 2.0, B2B Integration
Command of languages:
Good command of English
Client company profile:
Our client company is a new venture for one of the region's most established multinational company. Strong backing and commitment from the Shareholders
What they offer:
• Competitive Compensation Package and Incentive System (13th and 14th Month salary bonus + Annual financial bonus based on company goals)
• 3 days home office/week, flexible working hours
• Less Regulation, Fast Moving Agile Organization
• State of the Art Technologies and Capabilities relying on in-house developed Products and 3rd Party Cloud SaaS / PaaS
• Lots of room for career growth
• Efficient Business Operations, lean Processes and low level of Bureaucracy
• Regional ambition, potential international expansion ahead
