A Tech People egy európai munkaerő-közvetítő vállalat, mely magasan képzett erőforrásokat, valamint kiszervezett technológiai szolgáltatásokat nyújt különböző iparágak számára.
Jelenleg az alábbi pozícióba keresünk munkatársat:
OPENSHIFT SZAKÉRTŐ + JAVA SPRING FEJLESZTŐ - KONTRAKTOR - BANK (GYK)
Munkavégzés helye: Budapest, IX. került, heti 3 nap home office
Fizetés: versenyképes
Munkavégzés formája: teljes munkaidős, hosszú távú, kontraktor
Start: akár azonnal
A pozícióról:
Megbízónk egy nemzetközi piacon is jelenlévő bank számára keresünk fejlesztő kollégát az alábbi elvárásokkal és feladatokkal.
Feladatok:
Open Shift kialakításának továbbvitele, szakértői feladatok ellátása, fejlesztő kollegák kérdéseinek megválaszolása (operator, logelemzés…)Devops/ alkalmazás lifecicle management feladatok ellátásaSpring Boot használatával Java fejlesztési feladatok csapatban (+tervezés, erőforrásbecslés )meglévő alkalmazás domain megismeréseegyeztetés analyst kollégákkal
Elvárások:
Devops/ alkalmazás lifecicle management tapasztalatok (szakértői szint)Open Shift/ Kubernetes magas szintű ismerete (szakértői szint)magabiztos Java Spring back end fejlesztői tudásfelső fokú végzettségcsapatjátékos, ügyfélcentrikus, megoldás-centrikus hozzáállásönálló, proaktív, felelős munkavégzésre való képességszakmai nyitottság, tanulásra való nyitottság dokumentációs szintű angol nyelvismeret
Előny:
OpenShift certificateA következő technológiák minél mélyebb ismerete: Windows, Linux, Spring, Soap, Jax-VS, IntellIJ IDEA, Maven, Jenkins, Sonar, Docker, GIT, CI/CDbanki tapasztalatangol szakmai nyelv ismerete
