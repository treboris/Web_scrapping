Currently we are looking for
AZURE CLOUD MIGRATION ARCHITECT (JD)
Location: Budapest, IX district (partly home office)
Salary: competitive
Type of employment: employee
ABOUT THE POSITION:
As the Cloud Architect for Migration you will be responsible for ensuring the Cloud adoption strategy by defining the cloud adoption roadmap aligned with the target platform solution design, business strategies and the existing landscape environment.
TASKS:
You will be responsible for creating the optimal migration journey by leveraging TSI standards as well as cloud native tooling and processes.You will be also contributing to creation and improvement of the migration scenarios, set of best practices and leveraging Zero Touch approach wherever possible. As Cloud Architect, it will be in your responsibility to manage the engagement with cloud service providers through the build and run phases to ensure that customer achieves maximum value from cloud initiatives and servicesImplement complex customer requirements in technical solutions in accordance with best practices, cost-efficiency and future security based on Microsoft AzureDevelop both High-Level and Low-Level design for all possible variations of migration or transformation to hybrid cloud products (private/public) based on AzureOptimize all onboarding scenarios for customers on private & public cloud together with 3rd party products for migrationCooperate with the product owner on the continuous development of the T-systems cloud adoption portfolioCollaborate with sales on preparation of complex customer inquiries and tendersBe an advisor for local and international customers on optimal cloud adoption strategies on AzureCollaborate directly with end customers, with T-systems partners and international teams
QUALIFICATIONS:
We are looking for an individual with strong background in public cloud technologies who is ready to help customers on their migration journey to public cloud. In addition, you ideally bring the following with you:5+ years of relevant work experience with Azure cloud technologiesA deep technical understanding of public cloud technologies and practical experience in public cloud adoption and migrationExperience with advocating and implementing Cloud Adoption Frameworks on Azure and other meaningful best practicesA can-do attitude which helps you to stand out in a fail fast culturePassion to work in a global and diverse organization and environmentExcellent presentation and communication skills (spoken and written English)Familiarity with application discovery tools and cloud migration software is an asset but not essentialExperience with Infrastructure as Code is an asset but not essential
