Currently we are looking for
AUTOMATION TESTER (E-G2)
Location: Budapest, I. district ( 3-5 days office)
Salary: competitive
Type of employment: employee
About the position:
Our partner is a Hungarian company developing an E-bike navigation application. We are now looking for manual and automation tester colleague to increase their team.
Key responsibilities:
Analyze, reproduce and track errors reported by the customer or by the usersRetest of the fixes provided by the development team, including regression testingManual test of Smartphone applications (Android, iOS) and software for dedicated devicesTest of internally developed computer tools and web applicationsField test, mainly with e-bikes.Design, write, maintain test cases and documentation (based on specifications)Perform exploratory testingWork in a team, collaborating with the other members of the Test Team, Developers and Project Management.
Qualifications:
Proven working experience in Software testing environmentSelf-motivated, with a proactive approach to work.Working proficiency in English.
Technical skills:
Programming languages, such as Java, KotlinExperience in web frontend automation (for example Selenium WebDriver, Selenium Grid)Relevant experience in using the Page Object Model design patternExperience in user interface automation on Android/iOS (for example Appium, Espresso, XCUITest)Clear understanding of the HTTP and the APIRelevant experience in API test automation (for instance REST-assured, Postman)
Preferred skills:
Ability to think critically, collect and analyze data, make decisions and solve problemsWillingness to learn and to develop your own skills to improve your performanceGood time-management, interpersonal and communication skills
