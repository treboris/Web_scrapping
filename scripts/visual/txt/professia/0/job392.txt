Information about the position Job description, responsibilities and dutiesPANTHEON .tech is looking for Java Software Engineers for project that involves SND technology. If this area is interesting your you, keep reading about our project below. Your job will involve working on a prototype project - on something, that no one has ever done before.PANTHEON .tech customer's application was developed to deliver intuitive and useful network applications that abstract away unnecessary complexities to efficiently manage end-customer enterprise networks. Such applications enable efficient provisioning of network services using a common set of operations regardless of underlying SDN technology, products, or network elements. This, in turn, enables customer's staff to quickly deliver requested network services compliant with architectural standards without navigating the nuances of vendor-specific network solutions or provide to the end customer the self-service function. This SDN portal solution includes firewall and switch automation that provides visibility to network and network security vendor's devices. You will make sure that telcos, data centers, or ISPs stay up-to-date on the latest technologies and in the end, contribute to a faster, more interconnected world.
