Informácie o pracovnom mieste Náplň práce, právomoci a zodpovednostiDo interného vývojového tímu hľadáme na dlhodobú spoluprácu Senior Backend Developera, ktorí má skúsenosti s vývojom webových aplikácií v PHP frameworku (ideálne Laraveli) alebo Node.js. Vytvárame fintech / insurtech nástroje novej generácie. 
Our stack consists of the following technologies:
Infrastructure - AWS cloud based on ECS Fargates, our internal team created IaC (Infrastructure as code) solution to automatically build and deploy our cloud stack. We use CDK for .NET to define PROSIGHT cloud infrastructure, Cognito for user authorization, s3 for storing and accessing objects, Parameter Store to enhance security and much more.
Frontend - We use React (Javascript framework) to ensure our customers have first-class user experience when using our software products.
Backend - Series of separate platform and application specific microservices written in PHP (Laravel framework). We have many new projects in pipeline, that's why Node.js is an alternative to existing PHP framework
API specification - Before writing a single line of code you are provided with API specification written in Swagger.
Versioning - We use Bitbucket and follow GitFlow workflow.
Agile - We deliver solutions bi-weekly, follow Scrum principles and use Jira and Confluence.
Design - We use Figma to create the design and integrate Figma files with Jira tasks.
Testing - We use Robot Framework for test automation.
When creating solutions we follow these principles:
SSOT (Single source of truth) - We pick resources that are shared across multiple apllications and create platform services to remove synchronization problems (e.g. the same identity with various emails, phone numbers across our ecosystem).
DRY (Do not repeat yourself) - Every piece of knowledge must have a single, unambiguous, authoritative representation within a system.
KISS (Keep it simple, stupid) - Break down the problem into small enough or understandable enough pieces, solve the problem and then code it, not the other way around. Zamestnanecké výhody, benefityFlexibilný pracovný čas.
