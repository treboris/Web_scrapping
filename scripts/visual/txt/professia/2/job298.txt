Informácie o pracovnom mieste Náplň práce, právomoci a zodpovednostiPre náš rastúci team hľadáme skúsených vývojárov na vývoj webu pre nemeckú automobilku.
Náplňou práce je vývoj web frontendu pre viaceré aplikácie. Vývoj na základe popisu v Figma /Postman/Swagger, spolupráca pri analýze požiadaviek, fixovanie bugov, code reviews, testovanie.
Pracujeme na viacerých projektoch, niektoré sú staršie zato už úspešne rozbehnuté, iné len teraz začíname.
Technológie a procesy ktoré pri práci používame:
- HTML, CSS, Java, JavaScript
- AEM
- Spring Boot
- REST APIs, JSON
- git, bitbucket
- code reviews
- unit tests
- CI/CD
- Akamai CDN
- Agile, Scrum
- Jira, Confluence
- MS Teams
- Mac OS, shell
Skúsenosti s vývojom frontend aplikácií v React/TypeScript a s Adobe experience manager sú vítané.
Pozícia je určená pre skúseného vývojára.
