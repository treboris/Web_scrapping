About the Role
You will participate in maintaining and enhancing the Python based automation framework (developed internally in Harman) and design solutions for interfaces, features, and improvements to work on cross platforms.
What You Will Do
Maintenance and enhancement of foundational simulation and test tool applications for the Python based automation framework
Work with stakeholders to understand requirements for testing hardware interfaces, features, and improvements
Collaborate and work closely with technical leads ensuring solutions work well with other layers of the global automation framework
Work with Architects and involve in designing, documenting, and implementing new cross-platform interfaces / features for generic simulations / test tool applications and libraries for use in the automated testing framework, ensuring solutions are modular, maintainable and scalable for use across OEM projects
Participate in POCs with new technologies and showcase to appropriate audience as well represent the Automation Team in customer demos as needed
Train / guide / initiate technically and kick-off of all global projects
Involve in maintaining software repositories and facilitate code review within a global development team
Active cooperation with organizational development and test teams to provide solutions that fulfill their automation, simulation, and tooling needs
Support on Python to existing .NET based framework extensions
What You Need
Bachelor’s Degree or higher in Computer Science, Electrical / Computer Engineering, or a comparable course of study
Minimum 5 years of experience in software development
Very strong Python programming skills
Expertise with threading / multi-threading, multi-processes-based programming
Experience with Git / version control, software development lifecycle and agile methodologies
Experience with Microsoft Visual Studio (or equivalent)
Understanding design patterns and architecture for SW product development
Basic familiarity with Linux based operating systems, embedded systems, communication protocols
General knowledge of automotive embedded systems, interfaces, and their applications
Proven and effective analytical, judgment, initiative, leadership, and execution skills
Experience in working with distributed development teams
Good English communication skills, both written and verbal; German is a desired
Exposure to cloud-based design for test automation framework
Experience developing and debugging for QNX / Linux / Android based embedded systems
Working knowledge of physical communication including I2C, RS232, SPI, or another serial interface
Experience with off-the-shelf or open-source test automation frameworks (e.g. Robot Framework)
Knowledge of build and deployment automation (e.g. Jenkins)
Experience with C#, .Net
