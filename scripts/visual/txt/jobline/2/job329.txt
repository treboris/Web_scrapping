Multinacionális cég - Senior DEVOPS mérnök - Azure cloud - Fluent English - Heti 1 nap iroda, többi HO
/Nagyrészt remote munka - Globál pozíció - DevSecOps mérnök - Audiotecnika, elektronikai gyártás/ Olyan munkahelyet keresel, ahol értékelik a képességeidet, tapasztalatod, lelkesedésed? Próbáld ki, mit érhetsz el ennél a cégnél, amely egyike a legnagyobb audiotechnikai és elektronikai berendezéseket gyártó nemzetközi vállalatoknak. A cég világszerte jelen van elektronikai termékeivel, vezető helyet foglal el az autó, mobil és telekommunikációs integrációs szolgáltatások területén. Az új kolléga egy nemzetközi Devops csapat tagja, a budapesti központban önállóan, egyedüli Devops-ként dolgozik majd. Feladatok:
A teljes CI/CD karbantartása - Bitbucket, Bamboo, Powershell, ACR, AKS, Cosmos DB
Kubernetes cluster, Orleans cluster és Docker konténerek karbantartása különböző környezetekben.
IaaS kód létrehozása, amely magában foglalja az Azure automatizált munkafolyamatokat, ARM sablonokat, Azure CLI-t és PowerShell szkripteket.
Szorosan együttműködni a biztonsági csapattal/FOSS-szal a PEN tesztelés, a Stress tesztelés és az infrastruktúra biztonság terén (10% a security feladat)
