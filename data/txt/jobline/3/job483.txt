Embedded developer
Tasks
Full cycle ECU (Electrical Control Unit) development, servo control supplement and operation system development in embedded C
Plan and execute embedded systems, prepare software requirements
Implementing and integrating requests coming from clients (automotive industry), internal electronics development and testing
Documentation considering security processes, delivering technical design, end-to-end software implementation, module level testing in C
Keep in touch with architects, software integrators and project leaders
Requirements
Electrical Engineer or IT degree (BSc/MSc)
On medior level 2-3+ years of embedded development background
Knowledge of analog and digital hardware for hardware-based programming
Knowledge of measuring tools for developing and testing
Fluent English is a must, German not required but nice to have it
Experience in MATLAB/Simulink/Python/AUTOSAR/MISRA C is an advantage
Engine control system or test system development background is also a plus
