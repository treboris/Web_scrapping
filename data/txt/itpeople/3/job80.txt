.NET FEJLESZTŐ (10715)Kórházi informatikai rendszereket fejlesztő parntercégünkhöz keresünk .NET fejlesztő munkatársunkat!Feladatok:• Egyedi informatikai rendszerek fejlesztése .NET / .NET Core környezetben,• üzleti logika implementálása,• fejlesztési dokumentációk készítése, karbantartása• közreműködés a fejlesztés különböző fázisaiban,• együttműködés a tesztelői csapattal.Elvárások:• Medior szintű .NET és/vagy .NET Core fejlesztői tapasztalattal és• objektumorientált szoftverfejlesztési ismeretekkel rendelkezik C# nyelven,• többrétegű architektúrák ismerete,• jártasság a tervezési mintákban és agilis szoftverfejlesztési módszertanokban,• tapasztalattal bír .NET technológiák használatában, szakértői szintű ismeretekkel rendelkezik C#, MVC, Entity Framework (Code First), SQL Server Transact SQL és LINQ, , jQuery, ASP.NET• többszálúság és a párhuzamos programozás erős szakmai ismerete• jártasság adatbázis -séma, táblázatok és eljárások tervezésében és fejlesztésében• jártasság web-szolgáltatások kialakításában.Készségek:• Problémamegoldó és koncepcióalkotó és valódi csapatjátékos,• csapaton belüli kommunikációs képességek,• precizitás és igényesség,• analitikus gondolkodás,• kiválóan érti az angol nyelvű szakszövegeket és videó anyagokat.Előnyt jelent:• Tapasztalat a webszolgáltatások fejlesztésében (WCF és/vagy Web API)• weboldalak (reszponzív) készítésében szerzett tapasztalat,• Windows szolgáltatások fejlesztésében szerzett tapasztalat• DevOps terület iránti érdeklődés és/vagy jártasság,• Single Page Application fejlesztési ismerete (Vue, React), valamint Blazor -- WebAssembly terület iránti érdeklődés és/vagy jártasság,• .NET Common Language Runtime (CLR),• JavaScript• GIT technológiák
