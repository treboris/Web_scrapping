Full-stack fejlesztő állások 
FULLSTACK FEJLESZTŐ (BACKEND FÓKUSSZAL)Partnercégünk 30 év nemzetközi tapasztalatával áll ügyfelei rendelkezésére a nemzetközi környezetben való egyedi szoftverfejlesztés és az informatikai tanácsadás területén. Dolgozóik magas elkötelezettségét bizonyítja az alacsony fluktuáció és a folyamatos létszámbővülés. A világ egyik legnagyobb digitalizációs nagyvállalatával közös, hosszútávú fejlesztési projektre keresik új kollégájukat:Fullstack fejlesztő (Backend fókusszal)Feladatkör:Szoftverfejlesztés nagyvállalati környezetben modern technológiákkal..NET Core backend fejlesztés.Minimális üzemeltetési és karbantartási feladatok.Használt technológiák: C#, .NET Framework (4-8), .Net Core 3.1, .Net 6.0, WebApi2, MS SQL Server (Entity Framework).Amire szükséged lesz:Legalább 3 év tapasztalat fullstack vagy backend fejlesztésben.Folyékony angol nyelvtudás írásban és szóban.Agilis módszertan ismerete (SCRUM).Előnyödre válhat:Big Data, HTML5, React/Angular, Azure Cloud és Clean Code ismerete.Amit partnerünk kínál:Stabil hátterű, több évtizedes múlttal rendelkező nemzetközi cég.Hosszú távú együttműködés.Remote munkavégzés lehetősége; igény esetén iroda Budapesten a 11. kerületben.Sok elemből álló cafeteria csomag.Érdekes feladatok nemzetközi környezetben és lehetőség szakmai fejlődésre.Jó hangulatú és támogató csapat.Ingyenes nyelvi képzések.Masszázs az irodában.Gyümölcs, kávé, tea, snack.Biciklitároló / Zuhanyzási lehetőség.
