Informácie o pracovnom mieste Náplň práce, právomoci a zodpovednostiPre nášho dlhoročného klienta hľadáme Java/React vývojára. 
Tvojou úlohou bude:
• realizácia server side FE vývoja (Java) a client side FE vývoja (Java/React) v spolupráci s Java BE vývojármi,
• vyvíjať nové, udržovať a modernizovať, 
• obrazovky/formuláre renderovené server side (java FE vývoj) alebo client side (java FE vývoj + React) spôsobom,
• prezentačnú logiku, mapovanie dát na back-end API, orchestráciu volaní API a Java front-end API pre React aplikácie,
• spracovávať dodané HTML šablóny od UI tímu,
• dodržiavať používané vývojové štandardy, vzory a postupy,
• implementovať riešenia tak, aby spĺňali požiadavky v oblasti škálovateľnosti, dostupnosti, výkonnosti a bezpečnosti,
• analyzovať a odstraňovať funkčné nedostatky softvéru,
• testovať (svoj) zdrojový kód a konfiguráciu,
• inštalovať a konfigurovať vyvinutý softvér. Zamestnanecké výhody, benefity• kurzy nemeckého a anglického jazyka,
• rôzne športové a relaxačno-regeneračné aktivity,
• 200 eur na vlastne športové a relaxačno – regeneračné aktivity,
• plne preplácané stravné lístky (len pri TPP),
• príspevok na DDS (po 12. mesiacoch) (len pri TPP),
• bonusy pri jubileách vo firme,
• bonusy pri narodení dieťaťa,
• firemné akcie,
• 2dni sick day ročne (len pri TPP).
Ponúkame tiež:
• atraktívnu, rôznorodú prácu na zaujímavých projektoch,
• kvalitné technické vybavenie, vyspelú IT infraštruktúru,
• možnosť profesionálneho rastu,
• motivujúce finančné ohodnotenie s možnosťou pravidelného prehodnotenia,
• možnosť vzdelávania a výberu rôznych školení (hard skill / soft skill).
