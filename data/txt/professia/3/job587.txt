Java Developer pre Digital IT (m/ž) V  Slovak Telekome neustále posúvame vpred tvorbu digitálnych produktov pre našich klientov. Okrem federálnych riešení (ČR/SR) vytvárame taktiež tie nadnárodné, pričom využívame agilné doručovanie automatizácie, DevOps a tvorbu nových riešení pomocou microservice architektúry. Do nášho Application Engineering (ktorý je súčasťou Digital IT) tímu, hľadáme Medior/Senior Java developerov, ktorí budú pripravovať samotné „srdce“ našich aplikácií, a to služby na báze microservices a ich implementáciu na harmonizované API. Konkrétne ide o mobilnú aplikáciu Telekom a taktiež všetky klientske aplikácie fungujúce na telekom.sk. Ak chceš participovať na riešeniach ovplyvňujúcich milióny klientov a zároveň pracovať s tými najnovšími technológiami, tak si tu správne. Čomu sa budeš venovať a koho hľadáme  U nás využiješ…  Java (min verzia 11, prechádzame na 17), (aktívna práca s Javou - min 3 roky sú podmienka)Java framework Spring (Spring Data/JPA, web/MVC, REST, Boot, Security, Cloud) a JUnit,Verzovací nástroj Git,Buildící nástroj Maven,Základy SQL – znalosť PostgreSQL databáz, ( skúsenosť s SQL aspoň rok je nutná)REST API a Swagger model,Kibana, Jiru, Confluence,Znalosť vytvárania mikro-služieb a ich architektúry,Jenkins, Gitlab,Angličtinu,Taktiež sa stretneš s …  Kafkou, Redis, Kubernetes (vyžadovaná skúsenosť s Kafkou, Redis a Kubernetes sú aspoň 3 roky),CDC nástroji Stream Sets a Informatica,Buildícím nástrojom Gradle,Thymeleaf šablónami,Java framework Hibernate,Aplikačným serverom JBoss a portálom Liferayi, Našim ponúkame to najlepšieSmartphone: Vyber si telefón a získaj naňho kompenzáciu každé 3 roky až do 900 €.Zamestnanecký paušál: Dátuj neobmedzene, telefonuj a SMSkuj za zvýhodnenú cenu na súkromné či pracovné účely.Bonusy a odmeny: Šikovnosť sa cení!
