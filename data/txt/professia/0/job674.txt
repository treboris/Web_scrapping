Microsoft Azure Data Engineer
Považuješ sa za “data enthusiast” a máš skúsenosti so službami Microsoft Azure? V tom prípade hľadáme práve Teba! Rozširujeme náš tím o skúseného Azure Data Engineera, ktorý má skúsenosti s vytváraním komplexných ETL a data warehouse riešení. 
Kandidát, ktorého hľadáme, musí byť schopný počúvať koncových užívateľov a následne navrhnúť a dodať bezpečné riešenie v rámci cloud alebo on-premise prostredia.
Práca podľa preferencie v jednej z troch lokácií: Bratislava, Košice, Banská Bystrica
Čo Ťa u nás čaká?
·	Návrh a vývoj dátových riešení na základe požiadaviek klienta a koncových užívateľov
·	Práca s novými technológiami v rámci medzinárodného tímu s top klientami
·	Neustále napredovanie v Microsoft Azure a iných industry technológiách
Čo na to potrebuješ?
·	Dlhodobé skúsenosti (minimálne 3 roky) s vývojom dátových riešení pre klienta alebo koncového užívateľa
·	Projektová skúsenosť a práca v Microsoft Azure
·	Znalosť Python, SQL, Node.js, C#, REST a iných relevantných programovacích jazykov, frameworks a rozhraní
·	Znalosť a projektová skúsenosť s komplexnými Azure data riešeniami a niektorými z nasledujúcich nástrojov:  
   - Azure Databricks, Azure Data Factory, Azure Synapse, Azure Analysis Services, Azure Data Lake, HDInsight, Power BI, SSIS, Event Hub, relational & non-relational DBs atd.
·	Skúsenosť s inými alebo doplňujúcimi big-data riešeniami:
   - napr. Kafka, Spark, Hive, Hadoop atd.
Výhodou:
·	Skúsenosti s Power Platform
·	Znalosť v oblasti DevSecOps a CI/CD 
·	Skúsenosti s rôznymi cloud a integračnými platformami (napr. AWS, Mulesoft)
·	Vytváranie AI/ML riešení
·	certifikácia DP-200/201
·	Agile mindset – schopnosť reagovať na zmenu priorít a požiadaviek, ownership, stakeholder management a kritické myslenie
·	Angličtina na komunikačnej úrovni (Nemčina navyše je výhodou)
