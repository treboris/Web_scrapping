Senior Application Developer - € 5,000 Sign-on BonusDeutsche Telekom IT Solutions Slovakia

                                Contract type
                            full-time
Information about the position Job description, responsibilities and dutiesPurpose
Develops new and maintain existing source code of  applications in programming language like Java, ASP.Net or C#  based on requirements of  internal or external customers. These applications support usually automation of business processes. Works independently or in project team on developing  and building  midsize and complex application for customer usage. Handles all phases of  software development  lifecycle by himself, especially for midsize applications, provides professional support in the field of deep system analysis, technical documentation, building related infrastructure for developed application by prototyping, research. Performs role of lead development architect, substitutes the role of  project manager in case of small or midsize software projects, serves as professional mentor.
General description
•	Develop new and maintain existing source code of applications
•	Works in SCRUM team on developing and building complex application solutions for customer DTAG
•	Serve as professional mentor for junior developers
For our projects, we are searching for developers within preferred stacks (Front end, Back end or Full stack):
• Tech. stack 1: Java, Spring Boot
• Tech. stack 2: Angular, React, VueJS
• Tech. stack 3: .NET, C#
Additional preferred knowledge/experience (as an advantage): any testing tool/framework, Kubernetes, Docker, Microservices, Git, cloud technologies (AWS, Azure..), DB...
Key accountabilities
•	Listen to user requests, consult with internal /external customers, project managers, assist in gathering requirements for preparation of application detailed feature specification as necessary basis of quality and functional application
•	Analyze of customer’s requirements and propose overall application / framework design of midsize application architecture (incl. interfaces, relation database structure), provide complex business analysis in order to build application functionality easy for maintenance and future extension according to customer needs as business intelligent solution that implements business processes in automated form
•	Provides project estimates, timeline like  total number of development hours, total number of testing hours and bug fixing and report it to customer for budget planning in advance
•	Implements customer’s requirements based on specification in selected object oriented programming language (JAVA, ASP.Net, C#)  into functional and user friendly application to meet customer’s needs
•	Provides testing of programmed functionality before deployment in order to avoid bugs and ensure quality of code
•	Implements security concept according to PSA policy in developed application to ensure accordance with DTAG standards
•	Write and update regularly complex technical  / system documentation (e.g. Authorization concept)  for success of penetration tests and also for official handover to operation process, also create analytical documentation incl. UML for other programmers and customer
•	Communicate project status and deliverables to managers, customer with efficiency and accuracy of any progress or delays to keep them informed and let them be ready for managerial decision and project’s adjustment  in advance
•	Maintain application on 2nd, 3rd level support by fixing bugs and regular check, programming updates of features on customer’s change request to satisfy customer needs and to ensure smooth running of the application with added value
•	Record spent time daily in transparent way to have basis for invoicing for customer and Controlling department purpose
•	Perform deploy on infrastructure, backup and restore procedures, handle release management in order to provide regular upgrades of software to have up to date features implemented according to agreement with customer
•	Other duties and responsibilities as assigned by higher management.
•	Manages partial project tasks assigned to him, timelines and communication
•	Mentor of junior developers
•	Provides technical trainings for basic users, junior developers (e.g., SharePoint sites administrators workshops) if required Wage (gross)3 100 EUR/monthFinal salary is negotiable.
