MICROSTRATEGY ADMINSITRATOR (LJT)
Location: Budapest, hybrid
Salary: competitive
Type of employment: full time, employee position
Tasks:
Administer and support (monitor and maintain) MicroStrategy servers (Intelligence Server, Web).Creating and maintaining user, user group, security filters, devices, transmitters, subscriptions etc.Assist developers in optimizing application performance.Assure all system setups, configurations and best practices are documented using standardized documentation.Troubleshoot issues identified by business and stakeholders.Implement and maintain security, manager server components and pro-actively monitor application resources.
Requirements:
BI Tools: MicroStrategy v9.4.1, v10.1, v10.2, v10.3 and v10.4Platforms: Windows 2000/XP Pro/2003 Server, Linux RHEL, Unix.Databases: Oracle, Teradata, MySQL, Hortonworks HDP, Access, Teradata Aster, MongoDB, Apache Hive, Microsoft SSAS, Microsoft SQL Server, Microsoft SQL Database and HP Vertica.Web Servers: Apache Tomcat, IIS, Oracle WebLogic and Red Hat JBossTroubleshooting and debuggingReport development : Execute SQL Queries against DB, understanding of MSTR architecture and componentsMSTR installation and maintenance on clustered environment on Windows, UNIX,LINUX environments
