Infrastruktúra mérnök (MS SQL)
A Sun City Software magyar tulajdonú, 10+ éve stabilan működő Microsoft Partner tanácsadó és IT szolgáltató cég, amely ügyfelei részére az alábbi szolgáltatásokat nyújtja:
Adattárház infrastruktúra tervezése, riportok fejlesztése, SQL szerver üzemeltetés
On-prem (Windows) és felhős (Azure) vállalati infrastruktúrák tervezése, implementálása, üzemeltetése
Oktatás, tanácsadás MS SQL és Windows szerver területen
A csapat nagyon tapasztalt szakértőkből áll. Olyan szakmai tudás és tapasztalat koncentrálódik itt, ahol mindig lesz mit tanulnod. A vállalati kultúrát a tanulás és szakmai munka jellemzi. Az motivál mindenkit, hogy magas színvonalon oldjunk meg kihívást jelentő feladatokat. Nem beosztottakat, hanem együttműködő munkatársakat keresünk, legyen az illető bármilyen tudásszinten. Tanácsadói attitűd jellemez minket: ügyfelekkel állunk napi kapcsolatban. A komolyság mellett persze van idő szóviccekre is. :) A visszajelzés nem csak fontos, de el is várt, hogy mindenki adjon és kapjon is - ebből lehet emberileg és szakmailag is épülni.
Cégünkhöz fűződik a HUG-MSSQL meetup és az SQLSaturday nevű konferencia sorozat, így aktív szereplői vagyunk a hazai adatos szakmai közösségnek.
Ügyfeleink jellemzően közép -és nagyvállalatok, ahol komplex, nagyszámú felhasználót kiszolgáló rendszereket építünk. Ez több százezer usert, több ezer konkurens felhasználót jelent a gyakorlatban.
Néhány tipikus projekt, amivel foglalkozunk, felsorolás szintjén:
MSSQL környezetek üzemeltetése: magas tranzakció (30K++ TPS) és felhasználószám (3-5K++ egyidejű) - Windows Server Failover Cluster üzemeltetése és üzembehelyezése, - SQL Server Availability Group tervezése, bevezetése és üzemeltetése, - SQL Server FCI tervezése, bevezetése és üzemeltetése, - SQL Server automatizálási feladatok, - biztonsági mentési stratégia kidolgozása, - query tuning, - T-SQL fejlesztések, - PBIRS, SSRS, SSAS, SSIS környezetek tervezése, bevezetése és üzemeltetése.
Konkrét projektek az utóbbi évekből:
Középvállalati szektor felső szegmensében lévő vállalat részére több országra kiterjedő ERP-hez tartozó infrastruktúra kialakítása
Szolgáltató cég számára hibrid szerveres infrastruktúra kialakítása
Nagyvállalat infrastruktúrájának migrálása Azure-ba és Azure-ról
Azure level 3 üzemeltetési támogatás nagyvállalati szegmensben
3-4000 konkurens felhasználót kiszolgáló rendszer üzemeltetése
240 ezer aktív felhasználót kiszolgáló rendszer megtervezésében való részvétel
