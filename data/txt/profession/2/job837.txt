AWS CLOUD MIGRATION ARCHITECT (JD)
Location: Budapest, IX district (partly home office)
Salary: competitive
Type of employment: employee
ABOUT THE POSITION:
As the AWS Migration Architect, you will be responsible for ensuring the Cloud adoption strategy by defining the cloud adoption roadmap aligned with the target platform solution design, business strategies and the existing landscape environment. You will be responsible for creating the optimal migration journey by leveraging TSI standards as well as cloud native tooling and processes. You will be also contributing to creation and improvement of the migration scenarios, set of best practices, and leveraging Zero Touch approach wherever possible. As AWS Migration Architect, it will be in your responsibility to manage the engagement with cloud service providers through the build and run phases to ensure that customer achieves maximum value from cloud initiatives and services.
TASKS:
Implement complex customer requirements in technical solutions in accordance with best practices, cost-efficiency and future security based on Amazon Web Services (AWS)Develop both High-Level and Low-Level design for all possible variations of migration or transformation to hybrid cloud products (private/public) based on AWSOptimize all onboarding scenarios for customers on private & public cloud together with 3rd party products for migrationCooperate with the product owner on the continuous development of the T-systems cloud adoption portfolioCollaborate with sales on preparation of complex customer inquiries and tendersBe an advisor for local and international customers on optimal cloud adoption strategiesCollaborate directly with end customers, with T-systems partners and international teamsCreate, define and improve processes and plans related to migration
QUALIFICATIONS:
We are looking for an individual with strong background in public cloud technologies who is ready to help customers on their migration journey to public cloud. In addition, you ideally bring the following with you:5+ years of relevant work experience with AWS cloud technologies and/or field relevant certification (Professional and/or Specialty)A deep technical understanding of public cloud technologies and practical experience in public cloud adoption and migrationExperience with advocating and implementing Cloud Adoption Frameworks on AWS, MAP, MRE, MPA and other meaningful best practicesA can-do attitude which helps you to stand out in a fail fast culturePassion to work in a global and diverse organization and environmentExcellent presentation and communication skills (spoken and written English)Familiarity with application discovery tools and cloud migration software is an assetSome experience with Agile tooling and processesExperience with Infrastructure as Code and automation possibilities (ex. Terraform)Experience with Version Control Systems (GitLab)
