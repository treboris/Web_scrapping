SCCM Engineer
What success looks like in this role:
Manage Distribution Calendar and Keep track of all Mass and Local deployments and maintain penetration reporting of all Mass and local Deployments.SCCM Asset Reporting and Reporting Software Usage, Maintain Deployment Calendar and License Validation process.Trouble Shooting on SCCM Distribution failures and Validating Packaged Applications for Deployment Readiness.Validating Software Deployment requests and deploying software for Individual and low client bases.Security Updates Deployment using Microsoft Configuration Manager and manage Patch Testing Deployments with change and Release ProcessImplementing service improvementsResponsible for proper escalation, communications, and management of production system problems.
You will be successful in this role if you have:
Software Deployment Process Knowledge of Microsoft Configuration Manager and good knowledge of Mass Software Distribution, Collection Creation and Managing Advertisements.Experience in Security Updates Deployment using Microsoft Configuration Manager and managing Patch Testing Deployments with change and Release Processes.Knowledge of SCCM client installation, software distribution, remote control, and other SCCM issues.Knowledge & worked on Microsoft Configuration Manager tools & utilities day-to-day operational stability Ability to prioritise multiple tasks.Knowledge of SCCM SQL Database Schema.In-depth knowledge of Client OS (Windows 10/11) and Troubleshooting.Knowledge of Windows servers IIS, Active Directory, DNS, etc.ITIL Foundation (optional), Certification in Managing Modern Desktops (Optional in case of good knowledge)Knowledge of other deployment tools like Intune, and Lan Desk will be an advantageKnowledge of other System center products will be added advantage.
